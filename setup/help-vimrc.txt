Plugins que eu uso:

    auto-pairs --> Auto complete pairs for vim
    nerdtree --> File explore
    ctrlp.vim --> Find files
    indentLine --> Indentline
    vim-airline --> Airline
    vim-airline-themes --> Airline themes
    vim-devicons (requer powerline fonts ou nerdfonts) --> Icons for terminal
    vim-molokai --> Theme for vim
    vim-pulyglot --> Highlight syntax
    vim-startify --> Home page and recent files for vim
    vim-transparent --> Set transparent backgroud for vim
